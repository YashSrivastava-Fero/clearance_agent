# clearnace_agent
HS code matching and related compliances
----------

**Required libraries (install via pip):**

pandas
sentence-transformers,
sklearn,
numpy,
re

-----------

**Input files:**

hs_code_detail.csv: Contains HS codes, descriptions, and import duties.

compliance_rule_book.csv: Lists compliance rules, agencies, and requirements.

-----------

**Installation**

pip install pandas sentence-transformers scikit-learn numpy
